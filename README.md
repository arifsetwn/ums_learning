# UMS Learning App

Aplikasi pembelajaran berbasis Flutter yang dikembangkan sebagai bahan ajar mata kuliah **Pemrograman Mobile** di Program Studi **Pendidikan Teknik Informatika**.

## Deskripsi

Aplikasi ini merupakan contoh implementasi navigasi dan manajemen halaman dalam Flutter, yang mencakup:

- **Bottom Navigation Bar** untuk navigasi antar halaman utama
- **Navigator.push** untuk perpindahan ke halaman detail
- **Passing Data** antar halaman
- Implementasi **StatefulWidget** dan **StatelessWidget**

## Fitur Aplikasi

1. **Halaman Materi** - Menampilkan daftar materi pembelajaran
2. **Halaman Video** - Menampilkan video pembelajaran
3. **Halaman Quiz** - Interface untuk kuis/latihan
4. **Halaman About** - Menampilkan informasi pengembang (3 data mahasiswa)

## Teknologi yang Digunakan

- Flutter SDK
- Dart Programming Language
- YouTube Player Iframe (untuk video player)

## Cara Menjalankan

1. Pastikan Flutter SDK sudah terinstall
2. Clone repository ini
3. Jalankan command:
   ```bash
   flutter pub get
   flutter run
   ```

## Tujuan Pembelajaran

Aplikasi ini dibuat untuk membantu mahasiswa memahami:
- Navigasi dalam Flutter (BottomNavigationBar, Navigator.push/pop)
- Manajemen state dengan StatefulWidget
- Passing data antar halaman
- Struktur project Flutter yang terorganisir
